# 💰 DevFinance

Sistema completo de controle financeiro com autenticação de usuários, desenvolvido para estudo e portfólio.

## 🚀 Funcionalidades

* 🔐 Cadastro e login de usuários
* 🔑 Autenticação com JWT
* 🔒 Senhas criptografadas com bcrypt
* ➕ Adicionar transações
* ❌ Excluir transações
* 📊 Dashboard com gráfico de receitas e despesas
* 👤 Dados separados por usuário

## 🛠️ Tecnologias

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express

**Banco de dados:**

* SQLite

**Autenticação:**

* JWT (JSON Web Token)
* bcrypt
